# Getting Started With My AI Brain

## Summary

This starter project helps a new user turn TigerMemory into a personal AI brain:
capture useful notes, keep durable knowledge in Markdown, and ask an LLM to
answer only from local evidence.

## Goal

Set up a local Wiki that an AI assistant can read, propose changes to, and use
as evidence when answering questions.

## Current Plan

1. Run `tm init`.
2. Configure an LLM provider with `tm llm guide`.
3. Add a few real notes under `wiki/`.
4. Use `tm admin propose` to draft a reviewable Wiki page.
5. Use `tm ask` to answer from local evidence with citations.
