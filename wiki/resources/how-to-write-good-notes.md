# How To Write Good Notes

## Summary

Good notes are short, sourced, and easy for an AI assistant to retrieve. They
say what is verified, what is inferred, and what still needs checking.

## A Useful Note Contains

- A clear title.
- A short summary.
- The important facts.
- Source links or file paths.
- A line between verified facts and guesses.

## Avoid

- Secrets, tokens, passwords, or private identifiers.
- Vague notes with no source.
- Long dumps that should be split into smaller pages.

## Example Structure

```markdown
## Summary

One short paragraph.

## Verified

- Fact with source.

## Open Questions

- What still needs checking?

## Sources

- Link or local file path.
```
