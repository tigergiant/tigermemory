# Archive

## Summary

Archive holds material that is no longer active but still useful as context.

## What Belongs Here

- Completed projects.
- Outdated plans kept for history.
- Old references that should not guide current decisions.

## Rule

Archive pages should not be treated as current instructions unless a current
page links back to them explicitly.
