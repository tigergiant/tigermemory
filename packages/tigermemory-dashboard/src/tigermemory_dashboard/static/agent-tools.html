指标说明（点击展开了解更多）

1. 什么是命中？

简单说：你问 AI 一个问题，它给你的答案里能不能找到正确信息。数字越高越好。

  • 首条命中：第一个结果就是正确答案，要求最高但最重要；
  • 前三命中：前 3 个结果里包含正确答案，实用性很高；
  • 前五命中：前 5 个结果里包含正确答案。若为 0%，说明搜索功能已失灵。

2. 什么是响应时间？

AI 搜索一次要花多久，单位是毫秒（ms，1 秒 = 1000 毫秒）。大致分三档：

  • 100ms 以内：非常快，几乎感觉不到等待；
  • 100ms ~ 500ms：正常速度，日常使用没问题；
  • 500ms 以上：偏慢，可能网络不稳定或服务器繁忙。
