先从普通版开始

普通版已经能管理 Wiki、查本地证据、连接 LLM。等你用顺了，再打开多设备、实时记忆和更多工具连接也不晚。

普通版 / local

Python + Git + Markdown + SQLite。适合第一次安装，不需要 Docker 或 WSL。

高级版 / hybrid

接入实时记忆、MCP、多 IDE 和远程网关。适合已经想把记忆系统接进日常工作的人。

现在推荐
local + LLM
依赖最少，适合第一次安装和试用。
